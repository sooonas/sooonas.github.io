# sonas.github.io

Personal academic homepage built with Jekyll and the [Minimal Light](https://github.com/yaoyao-liu/minimal-light) theme.

## Quick Start

1. Edit `_config.yml` with your personal information
2. Replace `assets/img/avatar-placeholder.svg` with your photo
3. Edit `index.md` to update page content
4. Edit `_data/publications.yml` to add your publications

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Then open http://localhost:4000 in your browser.

## Deployment

Push to the `main` branch and GitHub Pages will automatically build and deploy.
